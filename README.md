# Bankist Application 😊

A web application designed professionally.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Context API

## Installation

1. Clone the repository

```bash
git clone https://github.com/MahyarCode/bankist-app.git
```

2. Navigate to the project

```bash
cd bankist-app
```

3. Install dependencies

```bash
npm i
```

4. Run the game

```bash
npm run dev
```
